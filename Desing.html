Etiquetas basicas HTML

El HTML hace referencia al lenguaje de marcado para la elaboración de páginas web

¿Qué es una etiqueta HTML?

Es un “markup languaje” lo que significa que está escrito con código que una persona puede llegar a leer sin necesidad de que este mismo haya sido copilado primero. En pocas palabras el texto en esta página web este marcado con estos códigos para así dar instrucciones al navegador web sobre como mostrar el texto.

Etiquetas HTML basicas

  • < body > para el contenido
  • < head > para información sobre el documento
  • < div > división dentro del contenido
  • < a > para enlaces
  • < strong > para poner el texto en negrita
  • < br > para saltos de línea
  • < H1 >…< H6 > para títulos dentro del contenido
  • < img > para añadir imágenes al documento
  • < ol > para listas ordenadas.
  • paraelementos dentro de la lista
  • < ul > para listas desordenadas
  • < li > paraelementos dentro de la lista
  • < p > para parágrafos
  • < span > para estilos de una parte del texto
